# 🌱 7-Day Beginner Coding Plan (Python)

This is my personal 7-day plan to start learning Python step by step.


## 🟢 Day 1 — Setup + First Code
**Goal:** Run your first program

```python
print("Hello, world!")

```

**Practice :** Change the message and Print your name

## 🟢 Day 2 — Variables & Input

Goal: Store and use data

name = input("Enter your name: ")
age = 20
print("Hi", name)

Practice : Ask user for age and Print " You are -----Years old"

## 🟢 Day 3 — Conditions (if/else)
**Goal:** Make decisions

```python
age = int(input("Enter your age: "))

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

**Practice:**
- Check if a number is even or odd

---

## 🟢 Day 4 — Loops
**Goal:** Repeat actions

```python
for i in range(5):
    print(i)
```

**Practice:**
- Print numbers 1–10
- Print your name 5 times

---

## 🟢 Day 5 — Functions
**Goal:** Reuse code

```python
def greet(name):
    print("Hello", name)

greet("John")
```

**Practice:**
- Create a function to add two numbers

---

## 🟢 Day 6 — Mini Project 🎯
**Goal:** Build a simple game

```python
secret = 5
guess = int(input("Guess the number: "))

if guess == secret:
    print("Correct!")
else:
    print("Wrong!")
```

**Challenge:**
- Add a loop (keep asking until correct)
- Add hints (too high / too low)

---

## 🟢 Day 7 — Challenge + Review
**Goal:** Practice and review

**Build one:**
- Simple calculator
- Password checker
- Multiplication table

**Also:**
- Review all previous days
- Fix errors and improve code
